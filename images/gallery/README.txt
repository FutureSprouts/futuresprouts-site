Drop event photos here (JPG, max ~1600px wide), then add entries to
galleryPhotos in config.js. Example:
  { src: "images/gallery/cleanup-2026-1.jpg",
    alt: "Volunteers bagging litter at the creek cleanup",
    caption: "Community Cleanup - Spring 2026" }
The Events page gallery appears automatically once one entry exists.
REMINDER: identifiable minors require a signed parent/guardian media release.
